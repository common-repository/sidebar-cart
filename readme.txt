== Description ==
The plugin will allow you to add Sidebar Cart on your website.

== Installation ==
1. WooCommerce Sidebar Cart required WooCommerce
2. Go to plugins in your dashboard and select “Add New”
3. Search for “sidebar-cart”, Install & Activate it
4. Go to WooCommerce -> Sidebar Cart settings page for setting options.
